Crystal Collector Game!

Let's Play!

You will start with a random number.

Choose a Crystal. Each Crystal has a specific value and by clicking you can collet points.

The object is to keep collecting gems and reach the magic number. Match it and you win! :) Go over and you lose. :(

You can't see the point value until you select it. We can't make it too easy!!

Think you have memorized the value of each gem? Tricky tricky, the value of each gem resets with a new random number at the start of each new game.
